# Chaoxing Demo

一个用于学习 Python、Selenium 与自动化相关技术的个人练习项目。

## 项目说明

当前项目仍处于非常早期的学习与实验阶段。

主要目的：

- 学习 Selenium 自动化操作
- 学习网页交互与 iframe 处理
- 学习 Python 项目结构
- 学习 Git/GitHub 的基本使用
- 尝试实现学习通相关功能

目前代码结构仍比较混乱，很多内容属于实验性质，并不是正式可用的 Demo 或完整项目。

## 注意

本项目仅用于个人学习与技术研究。
